# Contributing

## Development Setup

```bash
npm install
npm run dev
```

Default endpoints:

- MCP: `http://127.0.0.1:3322/mcp`
- UI: `http://127.0.0.1:3322/ui/`

## Pull Request Guidelines

1. Keep changes focused and easy to review.
2. Update `README.md` when behavior, configuration, or APIs change.
3. Do not commit runtime data, local environment files, or `node_modules/`.
4. Run `npm test` before opening a pull request.

## Commit Guidance

- Use clear, descriptive commit messages.
- Prefer one logical change per commit.
